SwellDatumPresenter

class SwellDatumPresenter : UIView, DatumViewPresenter

Datum presenting the Swell Data. Info label will desribe the current water temperature. Rings oscillating inside the circle ring indicate the period of the swell. Indicator triangles (as many as swells for the current hour) will scale relative to weight of swell.
